# PyEditFinalVersion
PyEdit IDE是一个基于PyQt6开发的现代化Python集成开发环境。它采用Material Design设计风格，为Python开发者提供了优雅的用户界面和高效的编码体验。项目完全用Python编写，支持跨平台运行，无需依赖Web技术，确保原生应用的性能和稳定性。  主要特性包括：智能代码补全（支持模块成员补全如time.sleep）、语法高亮（支持多颜色分类）、冒号自动缩进（智能检测字符串/注释避免误触发）、集成终端（支持pip/cd等命令）和代码执行功能。界面使用qt-material主题库，提供符合Google设计规范的现代化视觉效果。  项目专为需要快速编写、调试Python脚本的开发者设计，特别适合教学、原型开发和日常脚本编写场景。
